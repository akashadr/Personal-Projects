In Python programming language, a comment is anything after a '#' symbol.
Example: print("Hello World!") # This is a comment. Comments are not part of the command, it not gonna execute something. Multiline comments are also possible in Python, those are enclosed by triple double-quote symbols:
Example print("Hello World!") """ This is a multiline comment. """
